Cabeçalhos HTML

Em uma página web, pode-se exibir “Títulos HTML”, que são títulos ou subtítulos da página em questão.

Exemplo:

Esses cabeçalhos da página web são definidos a partir das tags <h1> a <h6>. Elas funcionam de maneira hierárquica. Logo, <h1> para definir o título mais importante (o título em destaque) e <h6> para definir o menos importante (Subtítulo).

Exemplo:

Os cabeçalhos são importantes para montar a estrutura da página, pois normalmente os usuários utilizam os títulos para folhear uma página.

Nota: Como os cabeçalhos HTML possuem um tamanho padrão, caso seja do seu interesse aumentar o tamanho da fonte, basta utilizar o atributo style, usando a propriedade font-size do css.
